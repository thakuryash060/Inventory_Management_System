# Medicine Inventory Management System

This is a Java-based system designed to manage medicine stock efficiently. It helps in maintaining records of medicines, tracking quantities, and handling manufacturing and expiry dates in a structured way.

## Features
- Add and update medicine records  
- Track stock quantity  
- Store manufacturing and expiry dates  
- File-based data persistence  
- Calendar-based date input for accuracy  

## Tech Stack
- Java  
- File Handling  

## Purpose
This project was developed for academic learning to understand how real-world inventory systems work. It focuses on applying Java concepts to build a practical management solution for small-scale pharmacy use.

## How to Run
1. Clone the repository  
2. Compile all `.java` files  
3. Run the main class
